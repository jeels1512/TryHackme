## What this path covers

DevSecOps integrates security into every stage of software development. This path covers:

- Secure Software Development Lifecycle (S-SDLC)
- CI/CD pipeline security
- Source code & dependency management security
- SAST and DAST scanning
- Container security (Docker, Kubernetes)
- Infrastructure as Code (IaC) security — on-prem and cloud
